# Image-Gallary-Hub
A responsive and interactive Image Gallery built using **HTML, CSS, and Vanilla JavaScript**.  
It includes smooth navigation, lightbox view, hover effects, and category-based browsing using images from Unsplash.

---

## 🚀 Features

- 📱 Fully responsive design  
- 🖼️ Grid-based image layout  
- 🔍 Lightbox view for enlarged images  
- ⬅️➡️ Next / Previous navigation  
- ✨ Smooth hover effects and transitions  
- 🧠 JavaScript interactivity  
- 🏷️ Optional category filtering  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  

---
👉 Images are either:
- directly placed in the same folder, OR  
- loaded using external image URLs
  I used here https://source.unsplash.com/
